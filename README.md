## Lista de Contatos

Uma Lista de Contatos, **desenvilvido em C**, onde o usuário pode realizar um CRUD.
Neste projeto, foi utilizado conceitos avançados de programação em C:

* Structs;
* Ponteiros;
* Alocação Dinâmica de Memória.
